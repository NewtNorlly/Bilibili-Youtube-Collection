## 📋 收藏夹简介

<div style="background-color: #fafcff; padding: 15px; border-radius: 8px; border-left: 4px solid #fb7299;">

本仓库系个人整理的 **B站（哔哩哔哩）+ YouTube内容收藏集合**，集中分类存放不同UP主、频道及主题的视频收藏、观看笔记与关联推荐。

目前已将个人日常收藏的知识类、演讲类、科普类等内容，按**UP主/内容主题**拆分到对应文件夹，全量托管至GitHub平台，采用「分类归档、持续更新」的方式管理，解决传统收藏夹“杂乱无章、查找困难”的问题，方便后续回顾、补充笔记。

依托GitHub的版本管理功能，可完整保留收藏内容的更新记录，清晰追踪内容的补充与整理过程，是个人知识收藏与内容回顾的云端资料库。

## 📝 收藏整理指南

一、分类原则

按UP 主 / 频道核心分类：每个文件夹对应一个特定 UP 主或频道，确保内容归属清晰

按内容主题辅助补充：同一主题下的跨 UP 主内容，可新增主题类文件夹统一收纳

二、内容记录规范

每个分类文件夹内建议记录：

视频链接：B 站 / YouTube 的对应播放地址（可标注平台）

观看笔记：核心观点、知识点总结（支持 Markdown 格式）

推荐理由：适合观看的场景、内容亮点

关联推荐：同主题的其他 UP 主 / 视频链接

三、查阅与更新方法

查阅：直接进入对应 UP 主 / 主题文件夹，查看记录的内容与笔记

更新：新增收藏时，在对应分类下补充内容；新增主题 / UP 主时，创建同名文件夹并开始归档

## 🚀 快速克隆仓库

克隆仓库到本地

```bash
git clone https://github.com/NewtNolly/Bilibili-Youtube-Collection.git
```

