1830年11月，在法国文坛默默无闻的<span style="background: #fff88f">司汤达</span>出版了他的长篇小说<span style="background: #fff88f">《红与黑》</span>，虽然只出版了1500册，可是直到司汤达1842年去世的时候都没有卖完，这样惨淡的销量，对于任何一个作家来说无疑都是一个沉重的打击，但是司汤达却坚信<span style="text-decoration: underline; color: #134C97;">“我将在1880年被人理解”</span>，同时他在自传性作品中这样说道：“我抽中了一张彩票，头奖是拥有一本在1935年被人阅读的书”。历史最终证明，经历了岁月的洗礼之后，人们逐渐意识到了司汤达及其作品的伟大之处，特别是他的代表作《红与黑》，在将近200年后的今天，依旧被全世界无数的读者所阅读、讨论和研究，而司汤达本人也因为其<span style="text-decoration: underline; color: #134C97;">开创了意识流小说、心理小说的先河</span>，被后人称为<span style="background: #fff88f">现代小说之父</span>，《红与黑》也被公认为<span style="text-decoration: underline; color: #134C97;">法国第一部批判现实主义的著作</span>，也是19世纪<span style="background: #fff88f">现实主义</span>的奠基之作。

大家好啊，我是3号，今天要与大家交流的书正是司汤达的《红与黑》。在上一期<span style="background: #fff88f">《包法利夫人》</span>的视频中，我曾经说过，每一个女人心中或多或少都曾经有过一个包法利夫人，正如<span style="text-decoration: underline; color: #134C97;">每个男人心中或多或少都曾经有过一个于连</span>。下面就让我们回到1830年的法国，去从《红与黑》主人公<span style="background: #fff88f">于连</span>的身上寻找自己的影子。

《红与黑》有一个副标题叫做“1830年记事”，为了更好地阅读这本书，我们首先要搞清楚1830年的法国是什么样的。简单来说，1789年至1794年的<span style="background: #fff88f">法国大革命</span>之后，天降猛男<span style="background: #fff88f">拿破仑</span>更是成为了当时无数进步青年的偶像，因为他本人就是一个<span style="text-decoration: underline; color: #134C97;">凭借个人奋斗实现阶级跃升的典范</span>，但是拿破仑在滑铁卢战役中失败之后，<span style="background: #fff88f">波旁王朝复辟</span>了。于连当时面临的困境，用不太恰当的比喻，就是一个出生于民国初期的知识青年，刚刚准备投身孙中山领导的国民革命，满腔热血地准备建功立业，而且这场复辟并不是一场只有百天的闹剧，而是<span style="text-decoration: underline; color: #134C97;">长达十几年的统治</span>，于连就是成长于这样一个<span style="text-decoration: underline; color: #134C97;">动荡、矛盾撕裂的时代之中</span>。

他小时候梦想着能够在拿破仑的军队中出人头地，成为一名万人敬仰的将军，可是还没有等到他长大，拿破仑已经是一个禁止公开讨论的名词了，当兵这条路显然已经是走不通了。聪明的于连敏锐地发现，在复辟的波旁王朝中，教职人员不仅可以手握重权，而且年薪很高，于是于连决定要去当教师，而这正是《红与黑》的第一层含义，<span style="background: #fff88f">红</span>代表拿破仑时代的<span style="background: #fff88f">红色军服</span>，而<span style="background: #fff88f">黑</span>则是代表着教师的<span style="background: #fff88f">黑色教袍</span>。虽然司汤达从未正面回答《红与黑》的含义，但是<span style="text-decoration: underline; color: #134C97;">这个解释是公认的正确答案</span>，当然其他意义的解读在后面我们还会继续聊到。

于连想当教师，但是教士也分三六九等，要想在这一行出人头地也不是一件容易的事情，不过于连有个特长，那就是他过目不忘的惊人记忆力，他为了博取当地<span style="background: #fff88f">谢朗神父</span>的欢心，把拉丁文的<span style="background: #fff88f">《圣经》</span>倒背如流，于连的虔诚感动了老神父，他也为自己博得了远近闻名的好名声，虽然他的内心深处根本就不相信上帝，他真正的信仰对象是拿破仑。于连的信仰其实就是司汤达本人的信仰，司汤达从小就很崇拜拿破仑，成年后参军跟随拿破仑四处征战，与此同时，他还是一个<span style="text-decoration: underline; color: #134C97;">坚定的无神论者</span>，他有句名言：“上帝之所以被人原谅，是因为上帝是不存在的”。

回到书中，于连虽然是个天才，但是在他的父亲<span style="background: #fff88f">索雷尔</span>看来，于连只能算是一个不务正业的废物，索雷尔是一家<span style="background: #fff88f">锯木厂</span>的老板，他需要的是身强体壮的劳动力，而不是只会读书的小白脸。有一天，索雷尔突然告诉于连，他可以滚出这个家了，按理说于连应该很高兴，因为他一直想要逃离这个<span style="background: #fff88f">原生家庭</span>，但是于连只关心一个问题：“我去了之后和谁一起吃饭？如果是去当佣人，我可不敢。” 不只是索雷尔，估计大部分读者听到这个问题也会一头雾水，为什么<span style="text-decoration: underline; color: #134C97;">于连最关心的是吃饭的问题</span>，正常人想的不应该是薪资待遇问题吗？

于连有这种奇怪的想法，源自<span style="background: #fff88f">卢梭</span>的<span style="background: #fff88f">《忏悔录》</span>，卢梭曾经到一个贵妇人家里去做客，贵妇人邀请他留下来吃饭，卢梭也答应了，但是之后卢梭才意识到，女主人是准备让他和佣人一起吃饭，卢梭就决定要离开，最后是贵妇人的女儿挽留，卢梭才答应和她们母女俩一起同桌吃饭。于连这么在意和谁一起吃饭，并不是瞧不起佣人这些底层人民，而是受卢梭的影响，<span style="text-decoration: underline; color: #134C97;">不愿意为不尊重有识之士的贵族服务</span>，所以直到他从父亲那里得到确认的消息，他才答应去做家教。

那个时候，我们称呼瑞纳先生的妻子为<span style="background: #fff88f">瑞纳夫人</span>，以表示身份，所以后面我们都只叫她瑞纳夫人。瑞纳夫人之所以要雇于连当<span style="background: #fff88f">家庭教师</span>，并不是为了孩子的教育，而是因为<span style="background: #fff88f">死对头</span>家里买了两匹好马，所以她要给家里配一个家庭教师，这样就可以<span style="text-decoration: underline; color: #134C97;">从面子上扳回一城</span>，但是对于瑞纳夫人来说，这并不是一个好主意，因为在她的印象中，<span style="text-decoration: underline; color: #134C97;">家庭教师都是粗暴的人</span>，动不动就会打小孩。

瑞纳夫人出身<span style="background: #fff88f">贵族阶级</span>，是家里的独生女，而且是一大笔遗产的唯一继承人，和包法利夫人一样，她从小就在<span style="background: #fff88f">修道院</span>里面接受教育，然后什么也没学到，走出修道院就结婚了，但是与包法利夫人不同，她并没有沉迷于什么爱情小说，也没有接触到什么新的思想，可以说是<span style="text-decoration: underline; color: #134C97;">脑袋空空的就嫁给了瑞纳先生</span>，也<span style="text-decoration: underline; color: #134C97;">未体验过真正的爱情</span>，稀里糊涂的就成了三个孩子的母亲。虽然她已经是一个30多岁的少妇，但是无论是身材样貌还是内心世界，其实都还是像一个青春期的少女，在她眼里，所有的男人都和自己的丈夫一样，除了金钱、地位、名声之外，对一切都麻木不仁。其实她的婚姻状况和同时代的<span style="background: #fff88f">安娜·卡列尼娜</span>是差不多的，但是她最后的悲剧形式又是和安娜完全不同的，因为这本书的主角不是瑞纳夫人，我就不多做分析和介绍了，大家如果感兴趣的话，可以翻看我之前关于安娜的解说视频。

当瑞纳夫人在自家门口第一次看见于连，并得知这个英俊文雅的少年就是新来的家庭教师，她之前的担心瞬间烟消云散，但是她还是当面问了于连：“先生，你真的懂拉丁文吗？你不会打我的孩子们吧？” 于连内心激动得不得了，因为这是他<span style="text-decoration: underline; color: #134C97;">第一次被贵族平等对待</span>，更何况还是一位温柔美丽的贵妇人，同时他又觉得夫人瞧不起他这个<span style="background: #fff88f">出身寒微</span>的穷小子，他的脑海中突然萌生了一个<span style="text-decoration: underline; color: #134C97;">抱住瑞纳夫人胳膊的大胆想法</span>。虽然我们作为读者可能搞不懂这是什么脑回路，但是于连不仅这样想了，而且还真的这样做了，这个举动把瑞纳夫人吓了一跳，但是她的内心不仅没有生气，甚至还有一点小惊喜。

之后于连就被带到了客厅，于连拿出了一本《圣经》，让三个孩子随便翻开《圣经》的哪一页，只要起个头，他就可以一直背下去，于连的这个<span style="text-decoration: underline; color: #134C97;">惊人之举让众人惊呆</span>，甚至连客人和所有的家仆都连连称赞。于连在之后的日子中，<span style="text-decoration: underline; color: #134C97;">把本职工作干得很出色</span>，和孩子们相处的也很好，但是每当瑞纳夫人对他表示关心的时候，他都会因为第一次见面的误会表现得十分冷淡，这让瑞纳夫人有点难过。

与此同时，瑞纳夫人的贴身女仆<span style="background: #fff88f">艾丽莎</span>不知不觉中爱上了于连，而且她最近继承了一笔<span style="background: #fff88f">遗产</span>，她觉得自己和于连挺般配的，就借着向谢朗神父忏悔的时候，表达了自己想要嫁给于连的想法，因为她知道，谢朗神父相当于是于连的父亲和老师，谢朗神父也很赞同这桩婚事，答应要替他去和于连说媒。艾丽莎忍不住和瑞纳夫人分享了自己的小秘密，瑞纳夫人得知后竟然像发高烧一样夜不能寐，她自己都不知道为什么，就是不希望于连离开这里，可是没过几天，艾丽莎哭着和瑞纳夫人说，于连拒绝了她和谢朗神父的好意，瑞纳夫人听到后立刻什么毛病都没有了，这种幸福的感觉让她大吃一惊，她不禁在心里问自己：“难道我爱上于连了吗？” 她<span style="text-decoration: underline; color: #134C97;">爱上于连却不敢承认</span>。

平静的生活就这样冬去春来，<span style="background: #fff88f">瑞纳夫人</span>和她的表姐一起，带着孩子们和<span style="background: #fff88f">于连</span>去乡下度假，于连在这里感受到了<span style="text-decoration: underline; color: #134C97;">真正的自由</span>，每天陪着两个少妇和孩子们读书、散步，仿佛自己才是这里的男主人，他和瑞纳夫人的关系也变得亲近起来。有一天晚上，他陪着两位夫人在树下聊天的时候，无意中碰到了瑞纳夫人的手，瑞纳夫人下意识地把手收了回去，于连突然间意识到，<span style="text-decoration: underline; color: #134C97;">不能让这只手缩回去</span>，否则如此自卑的自己怎么可能干成大事，成就<span style="background: #fff88f">拿破仑</span>那样的伟业呢？

第二天晚上，他像往常一样和两位夫人聊天，经过长时间的思想斗争之后，他突然抓住了瑞纳夫人的手，瑞纳夫人吓了一跳，把手缩了回去，但是于连态度强硬的重新抓住了她的手，瑞纳夫人挣扎了一会儿，最后<span style="text-decoration: underline; color: #134C97;">把手留在了于连手里</span>，再也没有缩回去。此后的时间里面，两个人就像什么都没发生，和平时一样闲聊着，期间瑞纳夫人起身把被风吹倒的花盆扶正，重新坐下之后，又<span style="text-decoration: underline; color: #134C97;">自觉地把手交到于连手里</span>，而这一切都是在表姐的眼皮底下偷偷完成的。当天晚上，瑞纳夫人沉浸在爱情的喜悦之中，激动的一晚上都没睡着，而于连只是因为<span style="text-decoration: underline; color: #134C97;">打赢了一场胜仗</span>，带着胜利的喜悦安然入眠。

于连因为得意忘形，沉浸式阅读拿破仑的书，甚至都忘了自己的本职工作，<span style="background: #fff88f">瑞纳先生</span>发现了于连旷工的行为，把于连好好训斥了一顿，瑞纳夫人怕于连一气之下辞职，就去安慰于连，并表示她丈夫并没有对这件事很上心，现在已经去组织佣人更换床垫了。于连听完立刻脸色都变了，他央求瑞纳夫人去自己的房间，把藏在床垫之下的<span style="background: #fff88f">小纸盒</span>拿过来，里面有一个头像，但是这是自己的秘密，瑞纳夫人千万不可以看。

于连之所以这么紧张，是因为那是<span style="background: #fff88f">拿破仑的头像</span>，在当时这可是一个重罪，而瑞纳夫人心里想的是，这一定是于连心上人的头像，虽然她心痛欲绝，但还是完成了任务，并遵守承诺，没有偷看盒子里面的头像。于连拿到盒子之后，立刻一把火把它给烧了，经历了这个风波之后，他反而是徒增了劫后余生的勇气，他对瑞纳先生说：“如果没有我，你的孩子会取得这样的进步吗？你竟然还来指责我，离开你，我照样有饭吃。” 瑞纳先生以为于连肯定是找到了下家，说不定要去自己的死对头那里当家教了，于是<span style="text-decoration: underline; color: #134C97;">主动提出给于连涨工资</span>，这一下子反而把于连给整懵了，然后在心里说：“这些有钱人真的是不能给他们好脸色看。”

另一方面，瑞纳夫人内心饱受煎熬，她既为自己爱上于连而自责，又因为怀疑于连另有心上人而痛苦，最后她决定要结束这段孽缘，可是她不知道的是，于连在心理上击败了她的丈夫之后，还顺便请了几天假。于连出门之前专门想要和瑞纳夫人道个别，结果却发现瑞纳夫人对自己冷若冰霜，于是一句话都没说，掉头就走了。当瑞纳夫人得知于连外出的消息，她的心都碎了，<span style="text-decoration: underline; color: #134C97;">忘却贞洁道德</span>，她知道心高气傲的于连肯定是另谋出路去了，自己以后再也见不到于连了。

实际上于连根本就没有想那么多，他只是到好朋友<span style="background: #fff88f">付凯</span>那里做客去了，付凯的生意头脑很好，挣了很多钱，一直劝于连来做自己的生意合伙人，并把自己稳赚不赔的商业思路告诉了于连，本来于连已经是心动了，但是认真思考之后，他还是<span style="text-decoration: underline; color: #134C97;">拒绝了付凯的邀请</span>，因为他觉得自己志不在此，如果整天为了生意东奔西走，自己永远都别想出人头地了，挣钱不是他的理想，<span style="text-decoration: underline; color: #134C97;">他想要成为拿破仑</span>。

在于连离开的日子中，瑞纳夫人因为害怕于连一去不返，而陷入了强烈的焦虑之中，等到于连回来了，她小心翼翼地问于连是不是要离开这里了，于连这个时候已经确定，瑞纳夫人是真的爱上自己，他的虚荣心告诉自己：“我一定要得到她，这样等我以后出人头地了，别人问我当年做家庭教师的经历，我可以说那是因为爱情才屈尊就任的。” 于是于连回答瑞纳夫人说：“我应该走的，可是我太爱你了，这对于一个年轻教师来说，实在是一个巨大的错误。”

于连的答案，使得瑞纳夫人在爱情的漩涡之中迷失了自我，晚上聊天的时候，甚至主动把手交到了于连的手里，可是他还没有高兴多久，于连就给他出了一个巨大的难题，于连告诉他，当天夜里两点钟，自己要到他的房间去。于连其实自己都搞不懂为什么要这样说、这样做，与其说是想得到瑞纳夫人的爱情或者身体，更不如说这是他给自己创造了一场战斗，而他一定要赢下这场战斗，为了证明自己的勇气，他抱着视死如归的决心，<span style="text-decoration: underline; color: #134C97;">夜里两点潜入瑞纳夫人房间</span>。在此之前，很多读者肯定会好奇这样一个问题：瑞纳先生就睡在隔壁，而且对于一墙之隔的事情毫不知情，那我只能说书里的剧情就是这么安排的。

于连进屋的时候，瑞纳夫人根本就没有睡，与其说他是在等待于连，不如说是在等待命运的审判，虽然他一开始拒绝了于连，但是当于连跪倒在他的面前，瑞纳夫人最后还是没有守住底线。当几个小时之后，心满意足的于连回到自己的房间，他在心里问自己：“所谓的爱情和幸福就是这么一回事吗？” 而瑞纳夫人还未从云雨之情中缓过劲来，就陷入了<span style="text-decoration: underline; color: #134C97;">深深的悔恨与自责之中</span>。

就在这个时候，瑞纳夫人的孩子生了重病，虔诚而单纯的瑞纳夫人认为，这是上帝对于自己罪孽深重的惩罚，她没日没夜地守在孩子面前，让于连远离自己，可是没过两天，又让于连回到自己的身边，因为没有于连的世界，对于他来说是更加可怕的地狱。好在孩子最终还是康复过来了，这场劫难使得于连和瑞纳夫人的感情更加热烈，特别是瑞纳夫人，<span style="text-decoration: underline; color: #134C97;">甘愿为于连下地狱</span>。

到了<span style="background: #fff88f">侯爵家</span>任职之后，于连还是故技重施，利用自己过人的记忆力和学识，<span style="text-decoration: underline; color: #134C97;">证明了自己的能力</span>，侯爵家可是真正顶级的贵族，于连在这里也只能算是一个秘书罢了，比佣人的家庭地位也好不到哪里去。侯爵夫妇有两个孩子，儿子叫洛尔拜，女儿叫玛蒂尔特，和于连都是同龄人，但是他们兄妹俩一开始都是有点瞧不起于连，直到有一天，洛尔拜主动邀请于连一起去骑马，虽然于连没有这么骑过马，但还是欣然接受了，结果当众摔了一个狗吃屎，晚宴的时候，于连却大大方方的把这件事情讲了出来，把玛蒂尔特给逗笑了。

侯爵觉得，一个乡下人能够在正式场合从容地讲述自己的倒霉事，<span style="text-decoration: underline; color: #134C97;">以后必定大有出息</span>，试用期几个月过去之后，侯爵基本上已经是完全认同于连了，凡是于连处理的文件，侯爵几乎是照批不误，但是另一方面，于连给侯爵家人带去的新鲜感也几乎消失了，所有人也只是把他当成一个<span style="background: #fff88f">能干的秘书</span>罢了。

说句题外话，很多同学读到这里可能会觉得莫名其妙，于连也好，外交官也罢，一言不合就要决斗，仿佛是一件很普通的事情，现实情况是在 1830 年那个年代，<span style="background: #fff88f">决斗文化</span>，尤其是<span style="background: #fff88f">荣誉决斗</span>，<span style="text-decoration: underline; color: #134C97;">曾是欧美上流社会男性文化中重要的一部分</span>，有很多我们熟悉的名人都参加过甚至热衷于决斗，比如普希金就在 1837 年的决斗中，腹部中枪不治身亡；美国第七任总统安德鲁·杰克逊一生中参加过多次决斗，其中在 1806 年与查尔斯·狄金森的决斗中，一枪打死了对方，自己胸部也中了一枪，离心脏只有几公分，差一点就极限一换一，这颗子弹终生留在了他的体内；同时代的米哈伊尔·莱蒙托夫、奥斯卡·王尔德、马克·吐温等著名文人，也是一言不合就要决斗，所以于连的决斗行为，在当时算是一个<span style="text-decoration: underline; color: #134C97;">正常的行为</span>。

于连二话没说就把梯子给放下去了，倒不是为了想要得到玛蒂尔特，而是为了<span style="text-decoration: underline; color: #134C97;">证明自己的勇气</span>，此时的他其实根本就不爱玛蒂尔特，甚至于他的口袋里面还揣着<span style="background: #fff88f">手枪</span>和<span style="background: #fff88f">匕首</span>，准备一旦遇到危险，就和玛蒂尔特同归于尽。令于连没有想到的是，玛蒂尔特是真的爱上了他这个穷小子，他真的打败了那些王公贵族，意外得到了玛蒂尔特的身体，此时的于连早已不是当初那个纯真的少年，他更多的是<span style="text-decoration: underline; color: #134C97;">心理上的满足</span>，而玛蒂尔特得到的却与自己的想象大相径庭，她本以为自己找到了一个像伯里公爵那样的英雄，更在于她好像也没有感受到于连对于自己的爱情，他在心里问自己：“难道于连不爱我吗？亦或者是难道我并不是真的爱于连吗？”

到了第二天，玛蒂尔特已经开始后悔了，她<span style="background: #fff88f">骄傲的自尊心</span>，不允许于连这样一个下等人成为自己命运的主宰，所以故意对于连爱答不理、冷若冰霜，现在反而是于连欲罢不能了，他理解不了玛蒂尔特如此巨大的转变，他本来并不爱玛蒂尔特，可是现在好像又爱她爱得不行。于连好不容易在图书室找到了和玛蒂尔特单独说话的机会，他直接问玛蒂尔特是不是不爱他了，玛蒂尔特直截了当的回答他：“我快恨死自己了，随便碰到一个阿猫阿狗就委身于他了。” 于连一听到玛蒂尔特说自己是阿猫阿狗，气愤的拔出了<span style="background: #fff88f">宝剑</span>，准备一剑杀了玛蒂尔特，可是玛蒂尔特不仅不害怕，反而昂首向他走过去，于连恢复了冷静，把宝剑放了回去，随后就离开了。

可是玛蒂尔特却久久的回味刚刚的场景，她感受到的不是害怕，而是兴奋，她又重新爱上了于连，她觉得只有于连这样的<span style="background: #fff88f">英雄</span>，才敢做出拔剑刺杀情人的举动，只有于连这样的人才配做自己的主人。之后的日子里面，她又开始主动找于连说话，这下又把于连给搞蒙了，不知道玛蒂尔特到底玩的是什么套路，殊不知，对于玛蒂尔特这样的女人，她追求的不是于连或者某一个具体的男人，而是<span style="text-decoration: underline; color: #134C97;">一种自己的想象</span>，就像我们之前聊过的包法利夫人，包法利夫人爱上的是爱情小说中那种完美情人，而玛蒂尔特的要求则更高，这个情人还必须是一位<span style="background: #fff88f">盖世英雄</span>，她在心里对自己说：“像我这样一个奇女子，献身给一个有价值的男人才是值得的，这样以后人们会说，当初我看上的不是这个男人英俊的外貌，而是他的雄心壮志。” 大家如果还记得的话，《了不起的盖茨比》中的<span style="background: #fff88f">黛西</span>也说过同样性质的话。

之后有一天，于连一时冲动，第一次向玛蒂尔特表明了自己的爱意，并表示自己如果没有玛蒂尔特的爱，就再也无法活下去了，正常恋爱中的女人听到情人的表白应该都是很高兴的，结果玛蒂尔特却大失所望，反而是瞧不起于连，她认为一个只懂得<span style="background: #fff88f">儿女情长</span>的男人，大概率是没有什么出息的，她又开始对于连不理不睬了。于连被玛蒂尔特这种反复无常的态度折磨得非常痛苦，相信自己是真的爱上了玛蒂尔特，其实我们读者都看得出来，他要的不是得到玛蒂尔特，而是<span style="text-decoration: underline; color: #134C97;">征服玛蒂尔特</span>，就像是要征服一个敌人，于是有一天晚上，他直接扛着梯子再次来到了玛蒂尔特的窗前，这次的风险比上次更大，因为他不知道玛蒂尔特会不会开窗，如果答案是否定的，他将失去一切甚至是生命，可是当玛蒂尔特看到从天而降的于连，立刻变得又惊又喜，觉得于连实在是太勇敢了，立刻投进了他的怀里。

玛蒂尔特不仅再次向于连奉献了自己的身体，还称于连是自己的<span style="background: #fff88f">主人</span>，而自己则是他的女奴，于连这个时候是彻底明白了，<span style="text-decoration: underline; color: #134C97;">要想征服玛蒂尔特，首先要征服她那不可一世的骄傲</span>，否则玛蒂尔特的爱情保质期不会超过一个星期。就在这个时候，侯爵让他参加了一个非常重要的秘密会议，仅仅是因为这个会议记录不能写在纸上，需要于连凭借其惊人的<span style="background: #fff88f">记忆力</span>记下来，然后再转达到需要的人那里去，会议的内容虽然很重要，但是时间有限，我在这里就不详细介绍了，留给读者朋友们自己去书中寻找答案。

于连在这次出差的过程中，遇到了一个<span style="background: #fff88f">亲王</span>，这个亲王是个情场高手，用现在的话说就是一个<span style="background: #fff88f">PUA 大师</span>，他看出于连正在为情所困，在他的追问下，于连说出了自己的困扰，但是并没有告诉亲王自己的情妇是谁，亲王给他出了一个主意，他让于连保持之前对于情妇的态度，但是却要<span style="text-decoration: underline; color: #134C97;">假装去追求她社交圈里的另一位女性</span>，具体的方法就是疯狂地写情书。于连表示自己做不到写情书给自己不爱的女人，亲王却说那都不算事，我这里有一本<span style="background: #fff88f">情书大全</span>，你直接拿回去抄就行了，里面的注意事项都写得清清楚楚。

于连一开始还对亲王的这套情书打法<span style="text-decoration: underline; color: #134C97;">不屑一顾</span>，结果到家之后却发现自己出差没多久，<span style="background: #fff88f">玛蒂尔特</span>几乎把他给忘了，甚至准备按照父亲的安排和一位<span style="background: #fff88f">公爵</span>结婚，于是于连决定反击，他选择的目标是一位<span style="background: #fff88f">元帅夫人</span>，这位夫人嫁给元帅之后，没多久就成了寡妇，同时是侯爵家的常客，特别符合亲王所说的特征。于连按照情书大全标注的顺序，开始给元帅夫人写情书，虽然元帅夫人一直没有回信，但是于连也无所谓，反正他都是原封不动地抄，甚至连自己抄的是什么都不知道，就在于连开始怀疑这个办法到底有没有用的时候，他收到了元帅夫人宴会的请帖，他一个小秘书，竟然<span style="text-decoration: underline; color: #134C97;">破格成为了元帅夫人的座上宾</span>，之后更是多次与元帅夫人亲密交往，这一切都被玛蒂尔特看在了眼里。

本来已经准备放弃的于连，现在竟然成为了元帅夫人的情人，这再次证明了自己之前没有看错人，于连果然是一个<span style="background: #fff88f">人中龙凤</span>，可是风水轮流转了，她现在想主动接近于连，可是于连却对她很冷淡，她想找于连说话，于连却对她不理不睬，她不知道的是，这一切都是于连<span style="text-decoration: underline; color: #134C97;">故意为之</span>，于连内心深处对于玛蒂尔特的回心转意其实是欣喜若狂的，但是他知道自己千万不能心软，否则玛蒂尔特对于自己的爱马上就会变质。

直到有一天，玛蒂尔特又看到了于连收到了元帅夫人的信，她再也受不了了，直接找于连摊牌了，她指责于连怎么可以这样对自己，自己可是于连的<span style="background: #fff88f">妻子</span>啊，玛蒂尔特打开了于连的抽屉，要看元帅夫人的信，结果却发现元帅夫人寄给于连厚厚的一沓信，于连甚至<span style="text-decoration: underline; color: #134C97;">连拆都没有拆开</span>，她这才意识到，于连果然不是凡人，于连不仅玩弄了元帅夫人，还根本就瞧不起元帅夫人，玛蒂尔特跪倒在于连面前，祈求于连的爱，于连冷漠地问她：“你如何才能保证你对我的爱不会再次改变了？” 玛蒂尔特表示自己愿意和于连私奔，这样自己就会身败名裂，没有回头路可以走，这就是自己的保证，即使这样，于连也没有答应玛蒂尔特，甚至还在故意和元帅夫人书信来往。

没过多久，于连终于是收到了玛蒂尔特让自己<span style="text-decoration: underline; color: #134C97;">心满意足的保证</span>，那就是玛蒂尔特怀孕了，而且玛蒂尔特坚持要告诉<span style="background: #fff88f">侯爵</span>，如果侯爵不同意，她就和于连私奔，如果侯爵要是伤害于连，她就为于连披麻戴孝。侯爵知道后，用乡下人都不用的恶毒语言，把于连骂得狗血淋头，表面上是于连和玛蒂尔特败坏了他的家族名声，其实更主要的是，侯爵早就计划好把女儿嫁给一个公爵，自己则可以利用这场联姻在<span style="background: #fff88f">政治上更进一步</span>，现在一切都成泡影了。

侯爵经过剧烈的心理斗争之后，最终还是选择了<span style="text-decoration: underline; color: #134C97;">屈服</span>，他给于连和玛蒂尔特一份年收入<span style="background: #fff88f">36000 法郎</span>的资产，并给了于连一份<span style="background: #fff88f">轻骑兵中尉</span>的任命状，于连可以说是心花怒放，他花了 6000 法郎购置了骏马和行装，才当了两天中尉，就已经开始想象着以后统领三军了，可是还没高兴两天，于连就收到了噩耗，<span style="background: #fff88f">瑞纳夫人</span>写了检举信，寄到了侯爵那里。瑞纳夫人在信里检举于连，说他又穷又贪、<span style="text-decoration: underline; color: #134C97;">虚伪狡诈</span>，专门引诱天真软弱的不幸女子，实际上都是为了钱。

于连看到信之后没有做任何辩解，因为他对于瑞纳夫人的笔迹实在是太熟悉了，这确实是瑞纳夫人写的信，他放下信，直接快马加鞭，回到了<span style="background: #fff88f">维利耶</span>，然后来到了教堂，当他再次见到自己曾经深爱的瑞纳夫人，她扣动了扳机，第一枪没有打中，然后补了第二枪，瑞纳夫人随即倒下了。开枪之后，于连并没有逃跑，反而是<span style="text-decoration: underline; color: #134C97;">冷静地接受了警察的逮捕</span>，直到被关进了牢房，他才意识到自己马上就要上断头台，但是他并不后悔，甚至当律师想为他开脱的时候，他直接说自己的谋杀行为是精心准备的，应该被判处死刑，之后，他的内心反而是进入了平静的状态。

直到狱卒告诉他，他的运气不错，瑞纳夫人并没有死，甚至伤得并不重，于连听完之后立刻泪流满面，他不是因为自己可以不被判死刑而高兴，而是单纯因为<span style="text-decoration: underline; color: #134C97;">瑞纳夫人活下来的消息</span>而高兴，他在一瞬间就开始对自己的罪行感到后悔，甚至因为瑞纳夫人活下来的奇迹而感谢上帝，在那一瞬间，他这个<span style="background: #fff88f">无神论者</span>发自内心地笃信上帝的存在。当站在生死的关口，他才意识到自己一直深爱着瑞纳夫人，而那段时光才是自己短暂人生中<span style="text-decoration: underline; color: #134C97;">唯一的美好时刻</span>。

我想很多读者到这里会感到很奇怪，为什么于连看到检举信之后会这么冲动，不是选择和侯爵以及玛蒂尔特辩解，而是要去杀死瑞纳夫人，其实答案就在那封检举信里，瑞纳夫人指责于连是一个又穷又虚伪、只会引诱不幸女人的下等人，而且他最大的目标就是为了钱，我们前面说过，于连是一个<span style="background: #fff88f">极度自尊又自卑</span>的人，而这封信恰恰出自他唯一真正爱过的人，这<span style="text-decoration: underline; color: #134C97;">彻底摧毁了他内心深处的那段美好情感</span>，他感受到的不仅是侮辱，更是背叛，所以才采取了这种极端的公开杀人方式，而之后的悔恨，依然源自他对瑞纳夫人的深切爱意。

玛蒂尔特得到消息后，立刻动用一切关系去营救于连，当权者保证只要于连配合，陪审团不会判处于连死刑，另一方面，玛蒂尔特又在内心感慨，自己的丈夫是多么的了不起，这种<span style="background: #fff88f">贵族式的复仇</span>，简直是自己的祖先伯里公爵在世。当她去探视于连的时候，甚至想象着于连的脑袋被砍下来之后，自己将会如何亲吻它，但是于连却劝玛蒂尔特把孩子生下来之后，交给瑞纳夫人抚养，然后自己改嫁，于连相信玛蒂尔特的热情维持不了几天，用不了多久，她就会把自己和孩子忘得一干二净，而玛蒂尔特不知道的是，于连根本就不打算博取陪审团的同情。

在法庭的辩护环节，于连发表了一段<span style="background: #fff88f">振聋发聩</span>的演讲，我个人认为，正是这段演讲，把《红与黑》的文学地位提升到了19世纪<span style="background: #fff88f">现实主义</span>奠基之作的高度，于连这样说道：“诸位，在你们看来，即使我的罪没有这么重，你们仍会借我来杀一儆百，以此来打击这样的年轻人们，他们出身低微，却有幸受过良好的教育，就妄想闯进你们所号称的上流社会，这就是我的罪过，而要审判我的，没有一个是我这个阶级的人，我不会向你们乞求任何怜悯，也不抱任何幻想，我犯了罪，定罪判刑是应该的，但是<span style="text-decoration: underline; color: #134C97;">审判我的人对社会的危害却要大出百倍</span>。” 于连的演讲可以说是<span style="background: #fff88f">死刑宣告</span>，上流社会可以容忍于连的杀人行为，却无法容忍于连公然揭开他们这个阶级以及这个社会的真相，于连最终被陪审团判处死刑。

当于连回到牢房，他并没有感到后悔，他唯一的遗憾是自己临死之前，无法再见<span style="background: #fff88f">瑞纳夫人</span>最后一面，但是他又在想，如果真的能够看到瑞纳夫人，他又该如何解释自己既想开枪杀她，又深深爱着她呢？可是当他一觉醒来，却发现瑞纳夫人真的站在了他的面前，瑞纳夫人是拖着<span style="background: #fff88f">病体</span>，克服重重困难才来到了这里，于连把这些日子中的思考以及对于瑞纳夫人的爱全部都说了出来，而瑞纳夫人则是告诉于连，他虽然中枪了，但是并不怪罪于连，甚至宁愿死在于连的枪下，他知道于连为什么要枪杀自己，因为那封检举信确实是他写的，但那是在<span style="background: #fff88f">教士逼迫</span>下写的。

得知真相的于连感觉自己没有任何遗憾，他从来没有像现在这样幸福过，他对瑞纳夫人唯一的要求，就是让她好好生活下去，因为刚刚瑞纳夫人说，自己的<span style="background: #fff88f">名声败坏</span>，自己的罪孽是如此深重，他希望能够和于连一起<span style="background: #fff88f">共赴黄泉</span>。

奔赴刑场当天，于连的内心非常的平静，因为他已经<span style="background: #fff88f">看清自己</span>，认识到自己之前是多么的虚伪，为了虚无的东西，忘记了生活，忘记了爱，所以才会那么的孤独，而面对死亡的他，正因为看清了自己的灵魂，<span style="text-decoration: underline; color: #134C97;">才会感到坚强和勇敢</span>。

于连死后，玛蒂尔特如愿得到了于连的<span style="background: #fff88f">头颅</span>，并亲手进行了安葬，瑞纳夫人在于连死后的第三天，抱着自己的孩子离开了人世。

读完《红与黑》这本书，每个读者应该都会思考这么一个问题，书名中的<span style="background: #fff88f">红与黑</span>到底是什么意思，正如我前面所说，司汤达本人并没有给出正面的回答，其中一个答案就是前面说的，红代表军队的红色军服，而黑代表教师的黑色教袍，由此引申的另一个解释是，红象征着<span style="background: #fff88f">拿破仑时代</span>与资产阶级革命，而黑代表着波旁王朝和封建统治，而我个人最喜欢的一个答案是，红代表着于连以及我们每个读者内心深处那些纯真的、善良的本质，而黑则是于连也好，我们读者也好，被职场、被社会中的虚伪、算计、潜规则所驯化后的<span style="background: #fff88f">黑化行为</span>。

于连是一个非常复杂矛盾的文学形象，而它最大的魅力，也正是源自于它的复杂性和矛盾性，在组织这一本书共读的过程中，依旧是有很多小伙伴不喜欢，乃至于讨厌于连这个人物形象，甚至可以说，几乎没有多少读者是发自内心喜欢于连，就我本人而言，我也是非常讨厌于连，不在于于连做错了什么，而在于<span style="text-decoration: underline; color: #134C97;">我讨厌那个像于连的自己</span>。

其实我们每个人都像于连一样具有<span style="background: #fff88f">两面性</span>，甚至是多面性，当我们合上书，仔细地分析一下于连，他到底做错了什么呢？有人说他是个凤凰男，欺骗女人的感情，说他是个野心家，说他是个虚伪的人，可是于连欺骗了谁的感情呢？直到于连去世，两个当事人女性还是依旧深深爱着于连，于连取得的所有荣誉都不是靠女人来实现的，原因很简单，当时的法国社会，女人是被排除在<span style="background: #fff88f">权力结构</span>之外的，真正帮助于连的都是教士和男性贵族，所以又怎么能批判于连是靠女人上位呢？

于连之所以最后身败名裂，走向毁灭，不仅不在于他的野心、虚伪，反而恰恰是因为他<span style="background: #fff88f">不够虚伪</span>，不够算计，如果他足够虚伪，那他就可以像瑞纳先生那样，头顶着绿帽子而无动于衷，如果他足够算计，他就不会选择玛蒂尔特，而会选择更有权势的元帅夫人，如果他不相信爱情，就不会因为一封检举信进而冲动杀人，如果他没有自尊，就可以向陪审团低头，而不是被送上断头台。

他是想要获得成功，像<span style="background: #fff88f">拿破仑</span>那样建功立业，但是他想要的是依靠个人的奋斗，而不是用别人的鲜血换取个人的声誉，于连对拿破仑的崇拜并非仅仅是对权力的渴望，更是对一种凭借才能而非出身获得成功的<span style="background: #fff88f">英雄时代</span>的向往，但是在当时的法国社会，这是不可能做到的，就像司汤达在另一本书中所说：“社会好比一根竹竿，分成若干节，一个人的伟大事业，就是爬上比他自己的阶级更高的阶级去，而这个阶级则想尽一切办法阻止他爬上去。” 而于连最后之所以慷慨陈词，毅然赴死，正是因为他在狱中<span style="background: #fff88f">认清真相</span>，<span style="text-decoration: underline; color: #134C97;">碌碌无为对他而言生不如死</span>。

于连身上的<span style="background: #fff88f">悲剧色彩</span>，开创了个人奋斗者这一文学形象的先河，福楼拜笔下的包法利夫人，菲茨杰拉德笔下的盖茨比，包括路遥笔下的高加林等等，其实都可以看到于连的影子，他们都是试图冲破自身阶级局限，最终被社会吞噬或异化的悲剧人物。司汤达的伟大之处就在于，<span style="text-decoration: underline; color: #134C97;">他没有把于连塑造成完美的殉道者</span>，于连的悲剧，很大程度上源自于他自身性格中的缺陷。

司汤达在《红与黑》全书的最后，写了一句意味深长的话：“献给幸运的少数人”，那么谁是书中幸运的人呢？什么样的人才能算得上是幸运呢？于连算得上是幸运的少数人吗？先说我的结论，我觉得于连是幸运的，虽然他的生命只有短短的20几年，可是他在面对死亡的时候却是坦然的，甚至可以说是没有什么遗憾的，“朝闻道，夕死足矣”，这句话可以用来形容于连最后的顿悟，就像我们之前聊过的毛姆的《刀锋》，于连就是那种<span style="background: #fff88f">跨越刀锋</span>的人。

于连出生在社会底层，但是他拒绝命运的安排，靠自己的奋斗努力，想要实现<span style="background: #fff88f">阶级跃升</span>，他的悲剧在于，最终他失败了，但是我们反过来想，如果他真的成功了，成为了一名贵族，那才是真正的悲剧，因为那就意味着他变成了自己原来最讨厌的人，但是我们又可以说，即使于连真的跨入了那个贵族阶级，他也不会满足，因为只要他的内心没有黑化，一旦时机成熟，他就会开始尝试<span style="background: #fff88f">毁灭阶级</span>。

我在组织共读的过程中说过这么一句话：“我讨厌于连，因为自己身上有于连的影子，我又佩服于连，因为我做的还不如于连。” 我想很多观看视频的同学，都是像我一样的小镇做题家，从小接受的教育就是好好学习，天天向上，一份耕耘，一份收获，于连相信一个底层人士凭借才能和勇气就可以征服世界，我们今天仍然信奉<span style="background: #fff88f">知识改变命运</span>，奋斗成就人生，可是当我们10年寒窗、披荆斩棘，越过高考这一关，进入大学、进入社会之后，发现现实不是这么回事，<span style="text-decoration: underline; color: #134C97;">付出未必得到回报</span>。

可悲的是，我们在嘲笑于连的同时，却没有勇气像于连一样去抗争，论学习能力和天赋，我们能像于连那样过目不忘吗？论工作能力，我们能像于连那样胜任多个岗位吗？现实情况是，我们很多人抱怨着社会的不公，却从未真正进行过反抗，在现实的困难面前不堪一击，我们嘲笑于连那不切实际的理想，可是<span style="text-decoration: underline; color: #134C97;">我们自己却早已没有理想</span>。

所以说《红与黑》是一本永远都读不完的书，我们今天依旧在阅读、讨论《红与黑》，绝不仅仅是为了读到书中人物的爱恨情仇，也不只是为了看到1830年的法国社会，更重要的是在于主人公于连面临的道德困扰、人生困境依旧是我们当代人时刻面对的难题，不知道各位读者是否从这本书或者这个视频中，找到了属于自己的答案。
