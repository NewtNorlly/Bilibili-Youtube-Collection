真正的<span style="background:#fff88f">自信</span>啊，不但是对自己抱有自信，同时对于自己做出的判断也抱有极大的自信。<span style="text-decoration: underline; color: #134C97;">自信是对自己及判断的信任</span>，信任朋友的人，首先是对自己的判断的信任，他也是<span style="background:#fff88f">勇气</span>的很重要的一个来源。我再说第二个情节，这是更让我觉得就像我们这样的普通人啊，就理解勇气必须要经过罗恩这一关。勇气、自信还源于一件事情，就是<span style="text-decoration: underline; color: #134C97;">接受平凡并为之自豪</span>。哎，他不是简单的说，哎呀，我承认我平凡了，我是个小人物了。不是这样，他能感受到平凡当中的力量，就是微尘的那个力量。大家还记得当时邓布利多呢给了他们三小只一人一个东西吧，一人一个神器是吧？法器罗恩拿的就是整个哈利·波特当中出现的第一个魔法物件吧，就那个<span style="background:#fff88f">熄灯器</span>。第二个应该是呃海格的摩托车吧，第一个就是他那个熄灯器。罗恩在出走之后，在他跟赫敏哈利·波特闹别扭出走之后，如果没有这个熄灯器他就回不来了。正是这个熄灯器让他又重回英雄的队伍。当时啊其实罗恩就有点沮丧啊，我记得罗恩就说，哎，你看这个邓布利多啊，他真是厉害，他就知道我呀肯定会意志薄弱，我呢肯定会打退堂鼓。哎呀，反正咱们三个里头啊，我是最需要这个激灯器的，因为他已经料定了我会退出啊。很多人也因此说，哎，你看罗恩就是他们三个里面最薄弱的一环，连邓布利多都料定了。但是后来赫敏说了一句话，我觉得是对的。<span style="text-decoration: underline; color: #134C97;">邓布利多知道罗恩会回来</span>，他就是我们普通人。

有一些人啊非常的自信，非常的有勇气。但他往往呢是害怕自己表现出懦弱，他害怕碰到那个平凡的自己，所以他要一直做有勇气壮这些人也很可贵啊，也是英雄。我举个例子啊，你比如说我最喜欢的一个人物就是<span style="background:#fff88f">小天狼星</span>，小天狼星怕过谁啊啊，那说实在的那真是小天狼星给死了。小天狼星要是不死就死活到这个最后大战的时候。那大家想想这个大军的领导人指不定是谁呢？小天狼心像一团火呀，在他身边的没有一个不被他温暖和点亮的。甚至我在这个阿斯卡班的囚徒那一本书当中，我都觉得这个小天狼星都快成一圣人了。就说话当中都冒着那种智慧的光辉，格言篓子凤凰社当中第一猛将，就他们这一代人当中，要说谁能豁得出去的，没有人比小天狼星厉害的，而且小天狼星是属于说我要是颓了，那我就颓了。但你要给我那么一丝丝希望，我上天追你凌霄殿，我下海赶你紫禁宫，我甭管你小矮星彼得跑哪儿去，我一定要把你逮着，我就是吃老鼠，我也得从这个阿兹卡班逃出来。<span style="text-decoration: underline; color: #134C97;">小天狼星是真摇滚真朋克</span>，对吧？但是我们以后聊这个小天狼星的时候，我可能也会细细的去说啊，我今天只是呃稍微弹一点点，那是因为他有<span style="background:#fff88f">存在主义危机</span>。小天狼星拥有如此之力量啊，他是特别害怕被那个<span style="background:#fff88f">布莱克家族</span>的黑暗所吞噬了，<span style="text-decoration: underline; color: #134C97;">害怕被家族黑暗吞噬</span>，他稍微不明亮一点儿，他就被那个布莱克家族的黑暗所吞噬了。那真的是夜空当中最亮的一颗星啊，不亵渎他天狼星的这个名字啊，这个这个我把它称之为叫做叫<span style="background:#fff88f">英雄的自信</span>。往往英雄起于草泽之间啊，都是要从黑暗当中散发光芒的。

但是你看<span style="background:#fff88f">罗恩</span>啊，罗恩从小其实被保护的很好，爸爸妈妈有爱，哥哥其实也都挺疼她的。罗恩虽然是排行老六，后面还有个妹妹，但是在她妹妹出生之前，她一直是家里的老嘎瘩呀，她是享受过全家人的宠爱的。虽然她的两个双胞胎哥哥经常跟她开玩笑，但是你能感觉到她跟他两个哥哥是很有感情的。你比如说罗恩，他不太会安慰哈利啊，你像这个哈利遇到什么事儿，哎呀，我头疼，哎呀，这个难受，赫敏会鞭策他啊，你要自律对吧？你要赶紧修炼你的大脑防御术了啊，你这样下去的话是不行的。罗恩不太会说这些，<span style="background:#fff88f">陪伴哈利</span>，甚至开一些看似不合时力的笑话，但这些东西都让哈利·波特非常的安心，<span style="text-decoration: underline; color: #134C97;">罗恩用陪伴给予哈利安心</span>，说笑话这一点一定是罗恩跟他的两个双胞胎哥哥继承的。总不能是跟婆西吧。虽然她一直用着<span style="background:#fff88f">二手的东西</span>，妈妈对她很严厉。但是你从她跟赫敏相处的这个方式上，你也能感受到，从她跟赫敏的身上看到了罗恩威斯莱先生和他的妻子的影子，说明<span style="background:#fff88f">父母的爱</span>在他身上是有很好的这个投射跟照应的。<span style="text-decoration: underline; color: #134C97;">罗恩未经历存在主义危机</span>，他可能没有真正的想过啊，我以后没有工作会怎么办啊，我会不会不被别人接受啊？就是我们很多读者看罗恩的时候，自己都很轻松很放松。罗恩跟哈利·波特闹掰的那一段啊，你别说哈利·波特觉得这个生活很压抑很难受，我们读者看着都很难受，很压抑。信要说你们什么时候和好啊？

<span style="text-decoration: underline; color: #134C97;">爱是自信和勇气的来源</span>。你在平时的生活当中有多么自然的不经意的生活在<span style="background:#fff88f">爱的氛围</span>里，那么你的勇气和自信流露出来的时候啊，你都感觉不到。就是<span style="background:#fff88f">罗恩·韦斯莱</span>他整个这个人他思考问题的逻辑他可能都没有学到，他一直是以爱的角度去思考问题的这就是维斯莱加人的幸福。比如说我讲一个细节啊，当时是在<span style="background:#fff88f">三强争霸赛</span>期间，<span style="background:#fff88f">小天狼星</span>担心哈利·波特嘛，他就化身成一只黑狗，以<span style="background:#fff88f">阿尼马克斯</span>的形态来到霍格莫德村要见哈利·波特。然后他就那个时候没有办法跟人家买东西吃嘛，他是通缉犯当时，所以他就变成狗，以后吃老鼠过日子。当时罗威斯雷听到这个消息之后，就跟哈利·波特说，哇，他一定非常爱你，哈利天知道他有多爱你。你看人家罗恩就没有说，哎呀，他是多么恨小矮星彼得。我看过很多解读，就是说小天狼星布莱克一定是恨透了小矮星彼得，所以他要生蛋如肉，食如止肉，饮如止血。没有人家罗恩没有这么去思考问题。你要知道啊，他是一直养这个老鼠，养了很长时间的，那个老鼠一直是他的宠物吧。就算后来他被恶心到了是吧，但是他是最有可能往那方面想的，就是这个小天狼星恨透老鼠，但他从来没有一点往那方面想过。人家就是特别能感知到这个爱，所以有的时候你听那种言论就特别有意思。就是说家里有孩子，你不要太给他太多的爱你不能让她觉得这个爱是理所应当的，这样他会不珍惜爱胡扯吗？那就是人只有生活在爱的环境当中，他才能<span style="text-decoration: underline; color: #134C97;">用爱的逻辑去思考</span>。而这种内心充足的状态，那种充盈的状态，是她勇气跟自信的重要来源。就是哈利·波特这套书啊，生孩子之前好好看一遍，<span style="text-decoration: underline; color: #134C97;">比育儿功能书更实用</span>。

就是这种安全感使得罗恩啊不知不觉的滋养出一个非常坚定的，不易被人攻破的<span style="background:#fff88f">精神内核</span>。她不怕自己犯错误，她不怕自己失败。甚至他在面对自己平庸的时候，他还能够勇敢地想象，我也要当<span style="background:#fff88f">级长</span>，我也要做<span style="background:#fff88f">魁地奇队长</span>啊，我也要怎么怎么样，对吧？当他和赫敏被选上级长的时候，哈利·波特都很羡慕啊。可是你觉得罗恩有什么“不配得感”吗？他会觉得哎呀我不配啊，我不配当级长。不是，他其实心里面特别高兴，特别安喜。他跟哈利说，哎呀，我还以为是你呢啊哈利只能客气一下吧。哈利说哎呀，估计是我犯的错误太多了吧，然后他马上就顺坡下驴了。他说嗯是的，估计就是你犯的错误太多了，就能感受到我。他就觉得嗯这就是我应得的啊，所以他才会吃醋。吃醋的人不是因为自卑，他吃谁的醋？他吃<span style="background:#fff88f">克鲁姆</span>的醋。克鲁姆是什么人呢？如果他的底层是像很多人讲的，他是个自卑的人，他会去吃克鲁姆的醋吗？即便他真颓了，他真的自卑了。因为在哈利·波特，在赫敏身边不自卑也难。确实有些怀疑自己了，他出走了。你看很多人啊做了一个错误的选择之后啊，他一辈子就被这个错误选择就困在这儿。很多人是这样的，这个密尔抹不开，哎呀，我不能低头造成一生的遗憾。在很多小说当中有很多人物就是因为这种，哎呀，我当年做了一个错误选择，我年幼的时候做了一个错误的选择，我我没有办法再面对自己了。你看罗恩，不是，罗恩马上就回来了，我回来了，嗨，我回来了。<span style="text-decoration: underline; color: #134C97;">安全感滋养坚定的精神内核</span>，<span style="text-decoration: underline; color: #134C97;">不自卑，认可自身价值</span>，<span style="text-decoration: underline; color: #134C97;">犯错后能及时回归</span>。

大家都记得纳威·隆巴顿拔出了<span style="background:#fff88f">格兰芬多宝剑</span>，但大家也不要忘记，当哈利·波特生死未卜的时候，当罗恩赫敏要面对单独销毁魂器的时候，是谁学了<span style="background:#fff88f">蛇佬腔</span>。在某种意义上讲，如果说“蛇佬腔”是哈利·波特<span style="background:#fff88f">伏地魔</span>某种象征的话，那么此时的罗恩就已经站在了跟他们一样的高度。拿着<span style="background:#fff88f">熄灯器</span>回来的罗恩，就是我们每一个普通人能感受到的最直接的那份勇敢，也是我们每一个普通人展现出来的让别人羡慕的那份勇敢。邓布利多经常说嘛选择很重要，不要选择那个轻松的东西。那么支持大家选择的是智慧吗？是不是我们要选择一个对的东西呢？其实很多选择不涉及什么对错，很多时候就是在黑暗当中去选择。甚至某种意义上讲，你选择什么都不能算错。<span style="text-decoration: underline; color: #134C97;">选择的关键是坚定坚持</span>，而不是选完这个之后开始犹犹豫豫，开始又怨天尤人。这 K 罗琳写出罗恩这样的普通人的勇敢和勇气，<span style="text-decoration: underline; color: #134C97;">普通人的勇敢更可贵</span>。在哈利·波特这部小说当中，有很多人和主角哈利·波特一样，都是有超凡脱俗的天赋的，但他们依然奋不顾身的向远方奔跑。但相比之下，罗恩维斯莱却是一个从理想回归现实的人物，在她年少的心中也曾有过不可抑制的激情，他也奋不顾身地追求理想，但最后的升华却使他坦然接受了自身的局限，双脚踩在坚实大地上的感觉给了的那份属于自己的勇气。或许许多人啊觉得这种姿态少了几分英雄的气概，可是我觉得啊，<span style="text-decoration: underline; color: #134C97;">直面平凡继续前行是另一种勇猛</span>。
