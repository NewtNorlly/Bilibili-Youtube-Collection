所以，按照之前的惯例，我还是要普及一点生物进化相关的常识，尤其是不相信进化论的短视频博主们可以先划走拉黑。<span style="text-decoration: underline; color: #134C97;">进化论根基是最短共同祖先理论</span>。什么意思？也就是说我们认为全世界的生物都源于一个最古老的<span style="background:#fff88f">原始祖先</span>，无性生殖大约在 40 亿年前出现。生物进化的研究呢就是研究从这个最早祖先一个个分裂出来到全球各种生命的<span style="background:#fff88f">谱系树</span>。那么任何两个物种之间我们都能寻找到它的一个最短共同祖先，也就是它们分裂的那个节点。

就在没人能解决这种争论的时候，进入上世纪 60 年代之后，我们知道了基因的本质 ——<span style="background:#fff88f">DNA双螺旋</span>被发现了。<span style="text-decoration: underline; color: #134C97;">DNA双螺旋揭示物种遗传编码</span>，我们意识到，原来所有的物种都拥有这种被称作核酸的化学物质来书写自己的性状编码，然后进行遗传。所以呢世界上各个物种的亲缘关系，我们就不再依赖通过人类观察归纳去进行分类了。甚至那些已经灭绝的物种，我们仍然能从遗骨中提取出基因去溯源。<span style="text-decoration: underline; color: #134C97;">基因可溯源灭绝古人类</span>，要说直立人灭绝太久已经找不到什么 DNA，但是这些尼安德特人，还有大荔人、金牛山人，离人类这么近的年代，通过<span style="background:#fff88f">基因溯源</span>已经不算太难了。那至于咋用基因溯源，可以打一个很有意思的比方哈。

而就在 2010 年，通过对克罗地亚尼安德特人的基因组草图测序，发现现代欧亚美人群都携带有 1% 到 4% 的尼安德特人基因，而非洲土著人群几乎不含有该成分。这就意味着智人在走出非洲的时候，和这些早期智人们没有产生完全的 <span style="background:#fff88f">生殖隔离</span>。<span style="text-decoration: underline; color: #134C97;">智人与早期智人无完全生殖隔离</span>。在 5 万到 6 万年前，一小部分人甚至和他们产生了杂交。这不仅证明了我们现在各地人类带有一些本地亚欧古人的特征是非常正常的，同时因为只有非洲以外的人携带了这些亚欧古人的基因，也更加证明了非洲起源论的正确性。<span style="text-decoration: underline; color: #134C97;">杂交证据支撑非洲起源论</span>。

而比起欧洲的研究，这次鲜明的主角 —— 我们中科院的 <span style="background:#fff88f">傅小妹</span> 更是贡献极大。她在对 <span style="background:#fff88f">东亚古人基因测序</span> 中也确定了，甘肃的夏河人，尤其是这次黑龙江的龙人，同样都属于丹尼索瓦人，并非是我们东亚人的祖先，他们也是我们的堂亲。我们中国发现的一些早期智人，很可能就是丹尼索瓦人，或者是和他们关系很近的族群。

但是最重磅的还是傅小妹的团队也惊喜地发现，我们现在东亚人群，尤其是藏族里面，都携带了丹尼索瓦人特有的高海拔适应性的 <span style="background:#fff88f">EPAS1基因</span>。傅小妹的研究发现，我们东亚人虽然主体属于非洲的晚期智人，但是体内不仅有尼安德特人基因，也有丹尼索瓦基因，甚至还保留了一种被称作 X 的未知的 <span style="background:#fff88f">古老型人属基因</span> 痕迹。<span style="text-decoration: underline; color: #134C97;">东亚人群含丹尼索瓦人基因</span>。至于这个古老人类到底来自于哪儿，是另一种早期智人，甚至还是更早的本地直立人？是已经发现了还是没发现？那还有更多的化石证据以及研究来验证。

傅小妹十多年的研究，不仅不像新闻所说推翻了非洲起源论，反而是一方面证实了 <span style="text-decoration: underline; color: #134C97;">智人非洲单一起源</span> 的同时，也深刻地发现了曾经那些第一次、第二次走出非洲的古人类，他们的后代并没有消亡，而是在演化历程中，并非和外来者势同水火、你死我活地把人家消灭，反而是一步步 <span style="text-decoration: underline; color: #134C97;">融合到了晚期智人中</span>。虽然占比很少，但也让我们身体里流淌着我们这片土地中古老血脉的一部分。

人类作为万物灵长最伟大的地方，就是作为一个遵循丛林法则、适者生存的动物，哪怕那种极端的 <span style="background:#fff88f">种族仇杀</span> 从未消失，但是希望天下大同的文明、希望全人类幸福的理想也从未消亡。

所以谈谈科学，我就想谈谈所谓立场等社会学的问题。

其实我一直很难理解，为什么总有一些人能把非洲起源说当成恶臭的 <span style="background:#fff88f">西方中心论</span> 一样，说成是西方世界对我们的阴谋和攻击。<span style="text-decoration: underline; color: #134C97;">非洲起源说非西方阴谋</span>。如果真是西方中心论，欧洲人为何不炮制一个欧洲起源论？非洲的地方在欧洲人眼里啥时候又成了西方了？难道在中国西边就是西方吗？作为曾经被欧洲人殖民过的地方，他们既然能承认一个非洲起源，为什么不能承认亚洲起源呢？

其次，且不说在美国大力压制中国的当下，我们现在的国策一直都是和一带一路沿线的非洲各国交好。但是哪怕诸位就是要对非洲人充满着欧洲极端 <span style="background:#fff88f">纳粹</span> 一样的歧视，情感上接受不了非洲起源，那咱也要明白，非洲起源说从来都不是指我们是如今非洲黑人的后代。<span style="text-decoration: underline; color: #134C97;">非洲起源非指黑人后代</span>。只是说我们所谓的黄种人、白种人、美洲人和非洲人一样，只是共同拥有一个曾经生活在非洲的共同祖先。

只要你还活在 21 世纪，你就绝对不会是任何一个族群的祖宗。我们都是在一个庞大的谱系树下分裂、交融，存续至今的后代。

但是我也能想到，让一群头脑只会正反单线思考的家伙理解复杂的谱系树，那可是比从猴子进化到人还有更大的挑战。

可能在一些人的意识里面，当祖宗是非常荣耀的事情。但是有人或许不知道，在以英德等日耳曼人为代表的 <span style="background:#fff88f">族群歧视</span> 观念里，他们认为最早开启大航海的西班牙人、葡萄牙人不如自己，最早开启文艺复兴的意大利人还不如西班牙人。整个地中海周边曾经无比繁荣的罗马帝国，如今成了落后的代名词。被认为是文明之源的埃及、美索不达米亚在他们眼里是邪恶的东方。至于印度、中国，会统统被贴上古老与神秘的标签，仿佛是个东方异世界一样遭到鄙视。

至于人类起源地的非洲，你知道他们是怎么看的？他们认为古老代表着落后，只有自己这群一千多年前刚从森林里面走出来、完全捡拾别的文明遗产并发展成如今世界霸主的家伙，才是真正的上等民族，甚至那个只有 200 年历史的美洲殖民地，那才是上等中的上等。

所以我实在是想不通，把东亚说成人类起源地怎么就能打击西方自信了？把我们好好的智人说成是早期智人，甚至说成是直立猿人，怎么就能提升我们的自豪感了？

甚至中国发现个恐龙、发现个古鱼，都能引起民族自豪。我实在不懂得，中生代、古生代的时候，我们所在的板块还没有拼合，那些肉鳍鱼与恐龙是咋把自己当成中国的？哦，对，这些人也不信 <span style="background:#fff88f">板块漂移</span>。

所以我觉得真要自豪，应该自豪的是像傅小妹一样，不是总想搞个大新闻，今天搞个 <span style="background:#fff88f">伪史论</span>，明天被封号了就暗戳戳地删视频，然后重新换个话题挑战人类学、继续收割 <span style="background:#fff88f">民粹韭菜</span> 的营销号，而是兢兢业业能做出扎实研究，也能包容不同声音，把 <span style="background:#fff88f">科学研究</span> 提升到世界前列的科学家。<span style="text-decoration: underline; color: #134C97;">自豪应源于扎实科学成果</span>。

各位哪怕真那么在乎民族祖先，真想对得起祖先，也绝不是天天口嗨，替他们 “赢” 就好了，而是能自己赢一次，他们才会真的欣慰。哪怕我们的同胞赢得了全世界，别人的功劳也不会跑到自己的身上。只有能在自己的领域问心无愧，挑战自我，做出自己的成果，才是属于自己的赢。

在这个世界上从来没有比真实更重要的立场。<span style="text-decoration: underline; color: #134C97;">真实是最重要的立场</span>。因为只要各位用小学水平的逻辑学也知道，建立在谎言上的立场，随时可以被推翻，那你如何维护你的立场？建立在煽动上的流量总会被大浪反噬，因为总有人比你更极端，没人能控制住<span style="background:#fff88f">民粹洪水滔天</span>。<span style="background:#fff88f">理性包容有爱</span>，才是属于人类的光辉。<span style="text-decoration: underline; color: #134C97;">理性包容是人类光辉</span>。否则，看起来再像人形，也不过是个会打字的猴子而已。

最后讲个小知识吧，英语的 “东方”（orient）和 “起源”（origin）都源于一个拉丁词根，本意是 “升起、开始”。因为东方是太阳升起的地方，也是一天的开始，但是或许在他们眼里，太阳最早升起也最早降落，他们西方才是未来。但不要忘了，昨天的太阳已经不会再升起。如果真要谈复兴，只要记得明天的太阳仍然会先从东方升起。夏天了，甭提龙人了，一起去黑龙江畔看看最早升起的太阳吧。
