印度最新民调实际上是因为<span style="text-decoration: underline; color: #134C97;">印度现实问题难以解决</span>。我知道很多人一说起印度就会想到强奸问题，两情若是久长时，又岂在朝朝暮暮。但实际上，强奸在印度的社会问题连前三都排不进去。

1985年，印度海关在集装箱里发现了1500具儿童骸骨，警方调查后发现，近期正常死亡的儿童没有这么多。于是警察迅速逮捕了集装箱的主人，审讯以后才得知，这个人就是专门买卖人骨的。<span style="text-decoration: underline; color: #134C97;">印度发生1500具儿童骸骨案</span>，<span style="background:#fff88f">人骨贸易</span>浮出水面。因为儿童骨骼过于稀少，又可以呈现骨结构发育的过渡阶段，所以更有价值。只要蒸煮、美白、浸泡以后，就可以卖出高价。

印度最高法院被迫出来表态说<span style="text-decoration: underline; color: #134C97;">禁止出口人体组织</span>，至于骸骨的来源就没有任何下文了，可过了几年以后，他们发现这个禁令根本没用，因为在人骨贸易国际市场上，印度完全没有竞争者。欧美医学院要人骨还是得从印度买，只不过从合法的变成了非法的，卖价更高了，西孟加拉邦的人骨工厂变得更隐蔽了而已。

2004年，印度和斯里兰卡的海岸遭遇了一场海啸灾难，夺走了无数人命，还造成了大量的灾民。很快，印度泰米尔纳都邦的海啸难民安置区就成了<span style="background:#fff88f">卖肾重灾区</span>，被当地人称为<span style="background:#fff88f">肾葬村</span>。因为印度政府没有做好灾民们的安置工作，灾民们穷困潦倒，急需钱用。而美国那边等待器官移植的人一直都排着长长的队，他们发现自己如果去印度接受器官移植，只需要美国二十分之一的价格就够了，两边的需求完全对上了。所以那些中介们只需要几百美元就可以买走难民们的肾。而肾脏进入医疗供应链以后，价格立刻火箭一般蹿升。作者采访到的一个移植中心告诉他他这边肾脏移植的价格一般介于35000至85000美元之间，每做一笔他的净利都在25000美元以上，实际能赚到多少得看手术地点。

还有<span style="background:#fff88f">卖血</span>、<span style="background:#fff88f">卖卵子</span>、<span style="background:#fff88f">卖孩子</span>。虽然不像人骨产业那样在国际上有统治地位，但但凡你能想到的人体器官交易，在印度都有庞大的市场，<span style="text-decoration: underline; color: #134C97;">印度高比例人口无基本人权</span>，如果你以为这就是极限的话，印度会告诉你，你的想象力还是太匮乏。

虽然印度很喜欢自诩文明古国，很喜欢说什么自己是世界上最大的民主国家，但<span style="text-decoration: underline; color: #134C97;">印度是松散资本主义联邦制共和国</span>。印度作为一个行政概念，本身都是被英国殖民者强行捏起来的，语言、民族和部落的问题一直都没搞明白，也缺乏一个占据多数的<span style="background:#fff88f">主体民族</span>。国家缺乏控制力，各邦自行其是，中央和法治向地方豪强低头，民众缺乏向心力，导致很多根深蒂固的问题，就算大家都看在眼里，但印度政府并没有能力解决。

拿咱们这边好理解的历史来对比的话，那就是许多旧中国留下了弊病，其实都是在建国很多年以后才得到解决的。比如多喝热水这件事，就是<span style="background:#fff88f">爱国卫生运动</span>以后才成为咱们这边的普遍现象的。南方多省普遍的血吸虫病等各类疾病也是这个过程中才消除的。<span style="text-decoration: underline; color: #134C97;">唯物主义为主是破四旧后形成</span>，更不用说一些偏远地区、落后地区的<span style="background:#fff88f">奴隶制度</span>、<span style="background:#fff88f">农奴制度</span>，都是在我们付出了牺牲之后才推翻的。我们中国人并不是生下来就唯物主义，觉得人人平等的。而印度既没有类似的<span style="background:#fff88f">社会主义改造</span>，也没有这样那样大规模的改革，中央政府甚至没有号令大家改革的能力。





















所以他们一直都觉得国家想要强大，必须靠<span style="background:#fff88f">抱大腿</span><span style="background:#fff88f">投机主义</span>，而不是<span style="text-decoration: underline; color: #134C97;">独立自主</span>。你可能会觉得不可思议，但对<span style="background:#fff88f">印度</span>而言，你们中国人觉得必须要独立自主才不可思议。

印度当初<span style="background:#fff88f">独立</span>的时候是1947年，他们有着几亿人口，有<span style="background:#fff88f">英国人</span>留下了<span style="background:#fff88f">工业基础</span>，独立过程没有太多战火，没把这些工业基础打烂，印度的起步条件其实比我们中国好很多，然而恰恰因为独立过程没有经历太多磨难，导致印度完全不理解<span style="text-decoration: underline; color: #134C97;">独立自主这四个字的含义</span>。

印度国家独立的过程中，有一个人叫<span style="background:#fff88f">钱德拉鲍斯</span>，他本来也是<span style="background:#fff88f">国大党</span>的，但和<span style="background:#fff88f">甘地</span>聊不来。第二年他就尝试<span style="background:#fff88f">武力抗英</span>，客观来说，他的武装反抗的理念是有可取之处的。可惜的是他的斗争失败了。因为他本着敌人的敌人就是朋友的原则，前后分别找<span style="background:#fff88f">希特勒</span>和<span style="background:#fff88f">东条英机</span>要援助去打英国人。这个思路现在看也很抽象，但站在当时被列强殖民的国家来说不算罕见，<span style="background:#fff88f">伊朗</span>的李萨汗也干了。在许多被压迫者眼里，第二次世界大战欧洲那边纯属狗咬狗，利用殖民者之间的矛盾也是一个思路。但结果显而易见，钱德拉也押错宝扑街了。钱德拉最终于1945年死于飞机失事。

1956年，一位卸任的英国首相访问印度的时候，对西孟加拉邦的官员说，正是钱德拉组建的<span style="background:#fff88f">印度国民军</span>，以及1946年印度的<span style="background:#fff88f">水兵起义</span>，让英国人彻底放弃了维持对印度殖民统治的希望。历史上<span style="background:#fff88f">武装斗争</span>道路的失败，甘地的成功，使得印度人时至今日依然很自然地认为<span style="text-decoration: underline; color: #134C97;">投机主义才是正确选择</span>。

我们中国人的观念则相反，是因为我们的整个<span style="background:#fff88f">近代史</span>就是一步步抛弃所有可能的幻想，直面淋漓鲜血的过程。<span style="background:#fff88f">保皇派</span>对封建君主抱有过幻想，结果被狠狠背刺。<span style="background:#fff88f">孙中山</span>对旧军阀抱有过幻想，结果被<span style="background:#fff88f">袁世凯</span>打脸。五四运动前夕，<span style="background:#fff88f">陈独秀</span>和学生们对美国总统<span style="background:#fff88f">威尔逊</span>抱有过幻想，以为对方会为我们主持公道，制止列强对中国的瓜分。当时学生们甚至去美国公使馆门前喊过威尔逊万岁。结果<span style="text-decoration: underline; color: #134C97;">巴黎和会的结果打醒了我们</span>。

<span style="background:#fff88f">蒋校长</span>对<span style="background:#fff88f">国联</span>、对所谓的国际调停抱有过幻想，总是期待外人能帮忙制止日本的侵略。结果对方从头到尾对他的政治表演连看都懒得看。英国人不是在绥靖，就是在拖后腿。<span style="background:#fff88f">共产党</span>也曾对蒋校长抱有过幻想，对非武装斗争的路线抱有过幻想，对左倾冒险主义的路线抱有过幻想，对照搬<span style="background:#fff88f">苏联</span>的法子抱有过幻想，对二战后的<span style="background:#fff88f">美国</span>抱有过幻想，对苏联老大哥抱过幻想。可结果是，这些幻想一次又一次地破灭在我们眼前，我们被逼着走上了一条最艰难、最鲜血淋漓、最痛苦但也是最伟大的一条路，那就是<span style="text-decoration: underline; color: #134C97;">坚持独立自主</span>。

我们的历史给了每一个中国人必须要独立自主的<span style="text-decoration: underline; color: #134C97;">共识</span>，我们甚至认为这个共识很稀松平常，但放在全世界来看其实很罕见。努力不一定有用，但<span style="text-decoration: underline; color: #134C97;">抱大腿是多数国家的现实</span>。

印度这几天又出消息说，<span style="background:#fff88f">莫迪</span>要在8月31号和9月1号这两天访华，如果真的来的话，那就是七年来首次访华了。你如果用中国人的思维看待，你可能会觉得人家都抛橄榄枝了，大概是深思熟虑了，可能要转向了，甚至投诚了。但实际上印度很多决定都是跟着大哥混，一拍脑门就做了，缺乏主体性，<span style="text-decoration: underline; color: #134C97;">依附性强</span>。
